# XiaoBao Editor

My browser extension to customize the usage of `tsinglanstudent.schoolis.cn`.


## Features

This extension is composed of two main components:

#### GPA & Performance Tools
- **GPA Webpage:** Puts a "Performance Tracker" button into the website's navigation bar. Opens a webpage that shows:
    - View your overall GPA for any semester.
    - Sees scores and grades for all your courses.
- **On-Page Score Popups:** When you view the details of a assignment, displays a small popup comparing your score to the class average and max score.

#### Font Editor
- **Custom Font:** Upload and apply any custom font (`.ttf`, `.otf`, `.woff`, etc.) to the website via the extension's menu.
    - Your chosen font is saved and automatically applied whenever you visit the site.  
- **Reset:** Instantly remove the custom font and revert to the website's default style.
Your chosen font is saved and automatically applied whenever you visit the site.

## Installation

To install this extension:

1.  Download this repository as a ZIP file and unzip it, or clone it using Git.
2.  Open Google Chrome and navigate to `chrome://extensions`.
3.  Enable **"Developer mode"** using the toggle switch in the top-right corner.
4.  Click the **"Load unpacked"** button that appears.
5.  Select the folder where you unzipped or cloned this repository.

The extension should now be installed and active.
